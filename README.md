# 🔐 Password Manager

A secure web-based Password Manager built with Python Flask, SQLite and encryption.

## ✨ Features

- 🔐 Master password protected vault
- ➕ Add passwords with site, username, category and notes
- 🔒 Passwords encrypted using Fernet encryption
- 👁️ Show/hide password toggle
- 📋 One-click copy to clipboard
- ⚡ Strong password generator
- 💪 Password strength meter
- 🔍 Search by site or username
- 🗑️ Delete passwords
- 🎨 Beautiful dark themed UI

## 🛠️ Tech Stack

- **Backend:** Python, Flask
- **Database:** SQLite
- **Encryption:** Cryptography (Fernet)
- **Frontend:** HTML, CSS, JavaScript
- **Version Control:** Git & GitHub

## 🚀 How to Run

1. Clone the repository
   `git clone https://github.com/pujanrasaili/password-manager.git`

2. Navigate to the folder
   `cd password-manager`

3. Create virtual environment
   `python -m venv venv`
   `venv\Scripts\activate`

4. Install dependencies
   `pip install flask cryptography`

5. Run the app
   `python app.py`

6. Open browser and go to `http://127.0.0.1:5000`

7. Master password is `admin123`
